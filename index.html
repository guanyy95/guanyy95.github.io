发表于 2021-10-25 分类于 Lua

Lua源码编译

1.下载lua源码解压缩

2. VS创建.lib静态库项目

Build Solution在工程Release文件夹中找到.lib文件，将文件拷贝到lua源码根目录下。

3.创建C++控制台Console Application

选择链接器–输入–附加依赖项–输入我们所编译的lua5.4.1.lib库文件名称(刚才生成后拷贝到Lua源码目录下的静态链接库)。然后点击应用，确定就好了。

在C++工程中创建Main.lua文件脚本

Runlua.cpp作为虚拟机启动脚本

跑起项目输出lua脚本内容
